# İkonlar

Bu klasöre Outlook Add-in için gerekli icon dosyalarını ekleyin:

- **icon-16.png** - 16x16 piksel
- **icon-32.png** - 32x32 piksel
- **icon-80.png** - 80x80 piksel

## Tasarım Önerileri

- Şeffaf arka plan kullanın (PNG formatı)
- Basit ve tanınabilir bir ikon kullanın
- Şirket logonuzu veya satın alma sembolünü kullanabilirsiniz
- Renk paleti: Mavi tonları (profesyonel görünüm için)
